# Bus-schedule
View the schedule of the selected bus stop.

# Install and Run
* `git clone https://github.com/pub-t/bus-schedule.git`
* `npm install`
* `npm start`
* `http://localhost:8080/client/`

# Publish project on surge
* `npm install --global surge`
* `surge`
